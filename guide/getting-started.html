Installation Steps

Follow these steps to install the extension

1

Enable Developer Mode

First, enable developer mode in Chrome to allow loading unpacked extensions.

  1. Open Chrome and navigate to chrome://extensions/
  2. Toggle on "Developer mode" in the top right corner
  3. You should see new buttons appear: "Load unpacked", "Pack extension", "Update"
2

Download Extension Files

Get the latest version of Element AI Extractor source code.

  1. Download the extension ZIP file or clone the repository
  2. Extract the files to a folder on your computer
  3. Ensure you have all required files in the elementsExtractor folder

Load the Extension

Load the extension into Chrome using the "Load unpacked" button.

  1. Click "Load unpacked" button in the extensions page
  2. Navigate to and select the elementsExtractor folder
  3. Click "Select Folder" to load the extension
  4. The extension should appear in your extensions list

4

Pin the Extension

Pin the extension to your toolbar for easy access.

  1. Click the extensions icon (puzzle piece) in Chrome toolbar
  2. Find "Element AI Extractor" in the list
  3. Click the pin icon next to it
  4. The extension icon should now appear in your toolbar

First Use

Test the extension with your first element extraction

üéØ Basic Element Extraction

Extract all interactive elements from a webpage

  1. Navigate to any webpage (e.g., https://example.com)
  2. Click the Element AI Extractor icon in your toolbar
  3. Keep default settings (Links, Buttons, Inputs selected)
  4. Click "üöÄ Extract Elements"
  5. View the results in the popup table

üî¨ Element Inspector

Inspect individual elements with detailed analysis

  1. Open the extension popup
  2. Click "üî¨ Inspect Element"
  3. Click on any element on the webpage
  4. View detailed locator information
  5. Copy locators to clipboard as needed

üìä Export Data

Export extracted elements in various formats

  1. Extract elements from a webpage
  2. Select export format (CSV, JSON, Excel, Table)
  3. Click "üöÄ Extract Elements"
  4. File will be automatically downloaded
  5. Open file in your preferred application
